项目启动
====
1. yarn
  1. yarn install
  2. yarn run dev
2. npm
  1. npm i
  2. npm run dev

目录结构描述
====
```
* node_modules                // 项目所需的依赖项   
* public                      // 公共文件  
    * favicon.ico             // 默认图标  
    * index.css               // 默认样式  
    * index.html              // 入口页面  
* src                                     
    * index.html              // 入口页面  
    * index.html              // 入口页面  
* App.vue                     // 页面入口文件  
* main.js                     // 程序入口文件  
* router.js                   // 路由管理  
* store.js                    // vuex  
* package.json                // 项目基本信息，包依赖信息  
* README.md                   // 项目说明文档  
* yarn.lock                   // yarn配置文件  
* .gitignore                  // git配置文件  
```
